# Brain-Games:
Brain-games â€” is a set of five console games that have to do with brain training. Each game has its own rules and questions, by answering 3 questions correctly you win. However, if the answer is wrong, the game automatically starts over again.**

** Catalog of games: 
* Even or odd number game.
* A game to determine the prime number.
* A calculator game to practice arithmetic.
* A guessing game for missing a number in a progression.
* A game of finding the greatest common divisor of two numbers.

## Installation manual:
1. Game directory installation:
```
make install
```
2. Game launch:
```
brain-even
```
